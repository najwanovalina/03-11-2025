Fungsi:
Menghitung total biaya pengiriman paket berdasarkan pada berat, jarak tempuh, dan volume paket
Memberi tambahan biaya jika volume paket melebihi batas 
Cara Kerja:
Program meminta pengguna untuk memasukkan berat paket (dalam kg)
Pengguna memasukkan jarak tempuh paket (dalam km)
Pengguna memasukkan dimensi paket (panjang, lebar, tinggi dalam cm)
Program menghitung volume paket dengan rumus:
volume = panjang × lebar × tinggi
Program menentukan tarif per kg berdasarkan jarak:
Jika jarak ≤ 10 km, tarif Rp 4250 per kg
Jika jarak > 10 km, tarif Rp 6000 per kg
Program mengalikan berat paket dengan tarif per kg untuk mendapatkan biaya dasar
Jika volume paket lebih dari 100 cm, maka ditambahkan biaya tambahan Rp 50.000
Total biaya pengiriman (biaya dasar + biaya tambahan jika ada) yang kemudian ditampilkan kepada pengguna
